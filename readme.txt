== Description ==

This plugins is build to calculate distance between two nearby locations. It can be useful for any property or travelling websites.
Plugin is tested upto wordpress version 4.5.1 and compatible with lower versions as well.

A shortcode [distance_calculator] can be used for adding calculator on front-end.

Provides responsive calculator and tested on cross browsers.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use shortcode on page or post to enable calculator
